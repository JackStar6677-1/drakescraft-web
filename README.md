# DrakesCraft · sitio web

Sitio estático del servidor **DrakesCraft** (Minecraft, temática Odisea, Slimefun y comunidad en español).

---

## Página en vivo

**[https://drakescraft.cl](https://drakescraft.cl)**

---

## Contenido del repo

| Archivo / carpeta | Descripción |
|-------------------|-------------|
| `index.html` | Inicio: conexión Java/Bedrock, lore, Slimefun, FAQ |
| `rules.html` | Reglamento |
| `store.html` | Tienda / donaciones (enlaces y avisos) |
| `styles.css` | Estilos compartidos |
| `script.js` | Menú, FAQ, tabs de tienda, copiar IP |
| `assets/` | Logo y recursos gráficos |

---

## Vista local

No hace falta build: abre `index.html` en el navegador o sirve la carpeta con cualquier servidor estático, por ejemplo:

```bash
npx --yes serve .
```
